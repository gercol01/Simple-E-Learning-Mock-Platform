What is Inheritance in OOP?

Inheritance is a fundamental concept in object-oriented programming (OOP) that allows a class to inherit properties and behavior from another class. In C#, inheritance is implemented through the use of classes and is achieved by defining a class based on another existing class, known as the base class or parent class. The class that inherits from the base class is called the derived class or child class. The derived class inherits all the members (fields, properties, methods, etc.) of the base class, including both the public and protected members.

Inheritance in C# provides several benefits:
1) Code Reusability: Inheritance allows you to reuse existing code by extending the functionality of the base class. The derived class automatically inherits the members of the base class, reducing code duplication.

2) Polymorphism: Inheritance supports polymorphism, which means that you can use objects of the derived class wherever the base class is expected. This allows for more flexible and modular code.

3) Hierarchy and Organization: Inheritance helps in organizing classes into a hierarchy, representing the relationships between different entities or concepts in your application.
